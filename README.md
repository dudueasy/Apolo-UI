# 工程架构:
Webpack + React + TypeScript + Jest & Enzyme 单元测试

### webpack 打包配置
通过 awesome-typescript-loader 编译并打包 tsx 文件

### Jest 单元测试配置
通过 ts-jest 编译测试用例文件

通过 enzyme 引入 React 测试环境
