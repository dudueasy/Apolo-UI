# 常用命令
### 编译项目
~~~
yarn build
~~~

### 执行测试
~~~
yarn test
~~~

### 自动部署
在项目根目录下执行以下命令 (需要确保当前用户对 deploy.sh 有可执行权限): 
~~~
./deploy.sh
~~~
